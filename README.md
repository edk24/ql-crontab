# 青龙面板脚本

此仓库包含一个丝路国际签到打卡脚本。

## 环境变量配置

### SLGJ_USER - 账户信息

#### 单账户格式
```
SLGJ_USER=phone=手机号&password=密码
```

例如：
```bash
export SLGJ_USER="phone=183014&password=Sl678"
```

## 功能特性

✅ 支持单账户签到
✅ 支持多账户同时处理
✅ 自动获取用户信息和钱包余额
✅ 签到结果以表格形式展示
✅ 批量处理统计信息
✅ 支持推送通知（青龙面板集成）

## 工作流程

1. 解析环境变量中的账户信息（支持多账户）
2. 为每个账户执行以下步骤：
   - 登录验证
   - 获取钱包余额
   - 执行签到
   - 记录结果
3. 生成统计表格展示所有账户的签到情况
4. 输出总体统计信息（成功数、失败数、总余额、总增加等）

## 定时任务配置

在青龙面板中添加定时任务：

```
任务名：丝路国际签到
脚本：python3 /path/to/main.py
定时规则：0,10 9 * * *  （每天 9:00 和 9:10 执行）
```

## 注意事项

- 多账户处理时，脚本会在账户间自动延迟 3-8 秒以避免请求过快
- 确保账户信息正确配置，手机号和密码必须匹配
- 支持与青龙面板的通知系统集成（PushPlus、Bark、Telegram 等）
